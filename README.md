# Gemini Runner

**Gemini Runner** is a fast-paced 2D side-scrolling runner game built with **React** and **Tailwind CSS**.
Run, jump, and dodge through beautifully themed worlds while collecting points, unlocking weapons, and customizing your character.

---

## Features
- Smooth Gameplay: Responsive and fluid side-scrolling action built entirely in React.
- Customizable Characters: Choose from multiple character selections
- Weapons & Power-Ups: Collect in-game weapons to help survive against enemy sprites.
- Scoring System:** Earn points by collecting coins and surviving longer.

---

## Gameplay
- Jump / Double Jump: Avoid obstacles and enemies.
- Attack: Use collected weapons to destroy threats.
- Collect coins: Increase your score and unlock cosmetics.
